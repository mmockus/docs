Jekyll

Getting Started#

https://jekyllrb.com/docs/installation/

Install Ruby#

sudo apt install ruby

Install Jekyll#

https://jekyllrb.com/docs/installation/ubuntu/

jekyll new .

open the gemfile#

code Gemfile

Modify Gemfile#

Disable the jekyll version it will be pulled from the github-pages

# gem "jekyll", "~> 4.2.0"

Enable github-pages and pin it to a version you can find the current versions of https://pages.github.com/versions/

gem "github-pages", "~> 215", group: :jekyll_plugins

Install theme#

Used the minimal-mistakes theme Configuration instructions: Configure minimal-mistakes

gem "minimal-mistakes-jekyll"# allow remote themes for Jekyllgem 'jekyll-remote-theme' # , '~> 2.0'# needed for minimal-mistakesgem "jekyll-include-cache", group: :jekyll_plugins

Useful commands#

bundle installbundle updatebundle exec jekyll serve
